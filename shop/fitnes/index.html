Главная /Магазин курсов/ Курсы фитнеса

ПРОГРАММА КУРСА:

1. Мотивационное занятие (1 занятие):

  • • Вступление, организационные моменты.
  • • О нас, и зачем нам все это надо.
  • • Разбор Вашей текущей ситуации.
  • • О других курсах, о нас, сравнение.
  • • Мотивация, деньги – как критерий измеримости.
  • • Список заданий + инструменты обучения.
  • • Заключение + напутствие.

2. Анатомия (3 занятия):

  • • Работа с клиентами, имеющими проблемы со здоровьем.
  • • Вода на тренировке.
  • • Принцип роста мышц.
  • • Декомпозиция тела.
  • • Свободные веса и тренажеры.
  • • Парни и девушки: отличия тренировок.
  • • Отличия тренировок: масса, сушка.
  • Кардио.

3. Составление программ питания (2 занятия):

  • • Для чего диета (три фактора)
  • • Эффект йо-йо
  • • 3 фактора Идеальной диеты
  • • Рацион вместо диеты
  • • Формулы расчета нормы калорий
  • • Дефицит/профицит
  • • Функции, нормы, типы белков
  • • Риски
  • • Калорийность, макронутриенты, микронутриенты
  • • Функции жиров
  • • Дыхательная функция
  • • Типы и нормы жиров, Омега-3
  • • Быстрые и медленные углеводы
  • • Несколько мифов
  • • Расчет для сушки и набора
  • • Анаболическая адаптация
  • • Вкусняшки

4. Маркетинг (4 занятия):

  • • Пояснение полного списка домашних заданий
  • • Упаковка, разбор группы по направлениям + комплектация страниц в социальных сетях
  • • Копирайтинг
  • • Составление маркетинговой картинки. Применение в таргетинге и на других площадках
  • • Разбор скриптов, возражений
  • • Инстаграм. Разбор. Упаковка
  • • Особенности для ведения Инстаграм
  • • Запуск рекламы + варианты
